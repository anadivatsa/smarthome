# Diagnose

Run `check_services.sh` first for any debugging or sitrep request, instead of re-deriving diagnostic commands from scratch.

```bash
bash /home/anadivatsa/smarthome/skills/diagnose/check_services.sh
```

It prints service status, disk, RAM, CPU temp, and the last 20 lines of logs for any non-active service — everything needed in one shot.
